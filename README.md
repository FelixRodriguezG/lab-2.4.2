# Lab 2-4-2: Tienda Opciones de Pedido

**Semana 2: 9 al 13 de Junio**

## Descripción

Este laboratorio consiste en crear una página web para realizar pedidos en una tienda (por ejemplo, una pizzería), permitiendo al usuario personalizar su pedido mediante diferentes controles de formulario. El objetivo es practicar el uso de menús desplegables, checkboxes, radio buttons y la personalización visual de formularios.

## Funcionalidades requeridas

- **Menú desplegable de tratamiento:** Sr./Sra./Otre (+ campo para ingresar nombre).
- **Campo de teléfono:** para que el usuario ingrese su número.
- **Desplegable de tipo de pizza:** al menos 6 opciones diferentes.
- **Checkboxes de ingredientes extras:** al menos 4 opciones a seleccionar.
- **Radio buttons para método de entrega:** 
  - Enviar a casa
  - Venir a buscar
- **Área de comentario:** campo de texto (textarea) para comentarios adicionales del cliente.
